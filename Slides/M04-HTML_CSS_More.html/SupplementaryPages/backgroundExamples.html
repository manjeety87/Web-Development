Background Examples

background-image: url("../images/canada.png");

opacity: 0.25;

background-size: cover;

All child elements will inherit the opacity so not the best choice for a page background; however, if you use opacity on a single element, it will only change that element (and its child elements)

background-image: url("../images/canada-transparent.png");

background-size: cover;

Image is made transparent before using ... see link in SLATE for easy way to make an image transparent

background-image: url("../images/canada-transparent.png");

background-repeat: repeat;

background-size: 30px 20px;

Repeats both on the x axis and y axis with a specified size

background-image: url("../images/canada-transparent.png");

background-repeat: repeat-y;

background-size: 30px 20px;

Repeats on the y axis (vertical) only
