SYST10049 / Accessing Library Resources
LinkedIn Learning, SkillPort, and O'Reilly Learning videos and e-books

LinkedIn Learning, Skillport, and O'Reilly Learning are free to you as a Sheridan student. In order to take advantage of these services, you MUST log in through the Sheridan web site (i.e. If you are asked for a Credit Card number, you have logged in incorrectly):

  1. Go to the Sheridan College Library Services home page
  2. Select Databases button
  3. Select the corresponding letter from the alphabet list:
    • L to search in LinkedIn Learning resources
    • S to search in Skillport resources
    • O to search in O'Reilly Learning resources
  4. Select the desired link. You may have to scroll down to find the correct link
  5. Use the Search function in libraries to find videos/books to use for reference.

You can also access Library Services directly from SLATE by selecting the tab at the top of the SLATE page. If you do not see Library Services, select the down-arrow on the More tab
