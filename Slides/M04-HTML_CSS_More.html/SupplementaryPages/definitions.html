HTML Hypertext Markup Language
HTML5 (the most current version of HTML) is not a programming language but a markup language; it provides the structure for a web page
Programming languages give your computer instructions whereas HTML is just a series of codes that tell the Web browser to display text/images/etc. in a certain way. These codes (called tags or elements) surround the content to be displayed.
CSS Cascading Style Sheet
While HTML provides the structure, CSS provides the formatting and layout for your site.
DOM / Document Object Module DOM is an arrangement of the content of an HTML document into a tree-like structure (the DOM tree). Each part of the HTML document is represented by a node on the tree including elements, attributes, and text.

Element/Tag/Attribute Somethimes element and tag are used interchangeably though there is a difference. An element normally consists of a start tag, content (optional), and an end tag which comprises an element structure (<p>HI</p>). A tag is just that; a single tag (<p>). An attribute identifies a property for an element and consists of an attribute=value pairing. Attributes always appear within an element's start tag (<img src='sheridan.png'> where src is the attribute and 'sheridan.png' is the value for that attribute).
Client-Server Network Simplified, a server is the host/hub that provides information or a service to a client (end-user).
Deprecated features You may see the term "deprecated" for certain HTML and CSS features. This simply means that the feature has been phased out. One example would be the <center> tag. It was replaced with the style "text-align: center;" when HTML5 was released.
Javascript JavaScript, often abbreviated as JS, is a high-level, interpreted programming language. It is a language which is also characterized as dynamic, weakly typed, prototype-based and multi-paradigm.
Alongside HTML and CSS, JavaScript is one of the three core technologies of the World Wide Web. JavaScript enables interactive web pages and thus is an essential part of web applications. The vast majority of websites use it, and all major web browsers have a dedicated JavaScript engine to execute it.
