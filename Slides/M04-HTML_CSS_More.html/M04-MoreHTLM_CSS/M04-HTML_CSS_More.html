    Review
    Meta tags provide metadata (information about other data) to the HTML document. Metadata is not displayed on the page but is parsed to retrieve the underlying definition or description.
    Further Meta documentation

    Browser Icon
    Icons appear in the tab of the browser page. Icons for the browser tab are linked in in the <head> section of your .html page.
    <link rel="icon" href="images/favicon.ico" type="image/png">

    Anchor tag options
    • Links to other pages within a web site
      <a href="htmPages/about.html">About Me</a>
    • Links to other pages on the web/external (target="_blank" will open page in a new tab)
      <a href="https://www.google.ca/" target="_blank">GOOGLE</a>
    • Bookmarks in current page
      <a href="#chapter3">Go to Chapter 3</a>
    • Email Creation (must have an email system installed/configured on your system for this to work - for Example: Outlook Mail)
      <a href="mailto:student@gmail.com?subject=Welcome%20Back">Email to Student</a>
      More on anchor tags
    • Other
      <a href="tel:9055551212" ...> will initiate a telephone call (used for Mobile sites)
      <a href="sms:9055551212" ...> will initiate a text message (used for Mobile sites)
      <a href="mms:9055551212" ...> will initiate a multimedia text message (used for Mobile sites)

    Background options
    • You can use the <body> tag to give the entire site a background color/image or can be used for any other container tag
      body { background-color: bisque; } /* bisque color background */
      section { background-color: rgba(255, 0, 0, 0.5) } /* 50% transparent red background for all sections*/
      Background Examples For Reference

    • Use the background-image style with options to include a background image
      body { background-image: url("back.gif"); }
      Use background-size: options and background-repeat: options to decide how image is displayed:
      background-size: 100px 200px;Image size specified
      background-size: cover;Scales the background image so that the smallest dimension reaches the maximum width/height of the element
      background-size: contain;Ensures that the entire background-image will display by showing the image at a scaled size
      background-size: 100% auto;Width is 100% and height is set to auto to keep aspect ratio
      ------------------------------------------------
      background-repeat: repeat;Image repeats both horizontally and vertically
      background-repeat: repeat-x;Image repeats horizontally only
      background-repeat: repeat-y;Image repeats vertically only
      background-repeat: no-repeat;Image is not repeated

    Colours
    • HTML Basic Colours
      AquaBlackBlueFuchsia
      GrayGreenLimeMaroon
      NavyOlivePurpleRed
      SilverTealWhiteYellow

    • Other Named Colours

    • Ways of determining colour:
      • RGB Percentages
        rgb (percent red, percent green, percent blue) / ex. rgb (40%, 50%, 56%); - medium blue colour
      • HSL (hue-saturation-light)
        hsl(#, %, %) / ex. hsl(0, 100%, 50%); shows base red
      • RGB Decimal Integer Settingsrgb (integerR, integerG, integerB) / ex. rgb(255, 255, 0); shows yellow
      • Hexadecimal Settings
        #RedGreenBlue; / ex. color: #69623D; shows desert green
      • Colour Picker for finding RGB and HSL values

    • Transparency - 4th parameter / Value from 0-1
      • rgba(255, 0, 0, 0.5) generates red with 50% opacity (50% transparency)
      • hsla (120, 100%, 50%, 0.3) generates green with 30% opacity (70% transparency)
